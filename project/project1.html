🌾 Crop Disease Classifier

A Convolutional Neural Network (CNN) model that classifies crop leaf diseases from images — helping Ethiopian farmers detect plant diseases early and protect their harvest.

Project Overview

Agriculture is a cornerstone of Ethiopia's economy, yet crop diseases cause significant losses every year. This project uses Deep Learning — specifically Convolutional Neural Networks — to automatically classify crop leaf diseases from photos taken on a smartphone, providing an accessible, low-cost diagnostic tool for smallholder farmers.

The model was trained on a dataset of 5,000+ labeled leaf images across multiple crop types (maize, tomato, wheat) and disease categories, achieving over 91% accuracy on the validation set.

Key Features

  • CNN architecture (ResNet50 transfer learning) fine-tuned on crop leaf dataset
  • Image preprocessing pipeline with data augmentation to prevent overfitting
  • Classifies 10 disease categories across 3 major crop types
  • Achieves 91%+ accuracy on the test dataset
  • Simple web interface for uploading a photo and getting instant diagnosis
  • Grad-CAM visualization to explain which leaf regions triggered the decision
  • Lightweight model exported for mobile deployment

What I Learned

This project deepened my understanding of transfer learning, CNN architectures, data augmentation techniques, and deploying ML models. I also gained experience with TensorFlow/Keras, OpenCV for image preprocessing, and creating explainable AI with Grad-CAM. Most importantly, I learned how to frame a real-world problem as a machine learning task.

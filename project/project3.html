🌍 ErosionNet – Soil Erosion Risk Prediction Using Deep Learning

A deep learning project that integrates RUSLE (Revised Universal Soil Loss Equation) with a feedforward neural network to predict and classify soil erosion risk across Ethiopian Woredas — achieving 96.21% validation accuracy.

Project Overview

Soil erosion costs Ethiopia an estimated 1.5 billion dollars annually, representing a 2–3% loss in GDP. Traditional empirical approaches like RUSLE are data-intensive and cannot scale for real-time, large-area monitoring. This project combines RUSLE-based target generation with a custom deep learning model (ErosionNet) to automate and improve erosion risk classification across Ethiopia's Woredas.

The dataset — Merged Woredas 75k.xlsx — contains 75,000 observations from five Woredas in the Amhara Region, North Showa Zone: Ankober, Menz, Merhabete, Minjar, and Kewet. It includes 18 terrain, climatic, and land-use features. Four Woredas (60,000 records) were used for training/validation and Kewet was reserved as the held-out test set.

This project is the first of its kind in Ethiopia to integrate a RUSLE-based equation with a deep neural network for soil erosion prediction, including a real-time satellite dashboard built in Jupyter Notebook.

🔮 Methodology

  • Data Preprocessing: KNN imputation for terrain features, median imputation for drainage density, one-hot encoding for categoricals, and feature scaling
  • RUSLE Factor Computation: A = R × K × LS × C × P — Rainfall erosivity (Hurni 1985), soil erodibility from soil types, LS from slope-length approximations, C from NDVI, P from land-use categories
  • Feature Selection: Pearson correlation, Variance Inflation Factor (VIF), and mutual information to eliminate collinear features; slope and rainfall identified as most important
  • ErosionNet (PyTorch): Feedforward network with 2 hidden layers, batch normalization, ReLU activation, dropout, Adam optimizer, cross-entropy loss, and class weighting for imbalanced data
  • Erosion Classes: Low, Moderate, and High erosion risk
  • Evaluation: Confusion matrix, validation accuracy, and generalization to held-out Kewet Woreda test set
  • Dashboard: Real-time satellite data integration with AI output compared to RUSLE values in a Jupyter Notebook dashboard

🏅 Results

  • Validation accuracy: 96.21%
  • Strong performance on the high-risk erosion class
  • Good generalization to the unseen Kewet Woreda test dataset
  • Slope and rainfall identified as the most important erosion-driving features
  • Successfully integrated real-time satellite data for live monitoring dashboard

💡 What I Learned

This project gave me deep hands-on experience with PyTorch for building custom neural networks, handling class imbalance with class weighting and synthetic data, applying physics-based equations (RUSLE) as feature engineering, and evaluating model performance with confusion matrices. I also gained experience in feature selection using statistical methods (VIF, mutual information) and integrating AI outputs with real-world environmental data for a live monitoring dashboard.

💬 Amharic Sentiment Analyzer

A deep learning NLP model that classifies the sentiment of Amharic text as positive, negative, or neutral — bringing modern AI capabilities to one of Africa's most widely spoken languages.

Project Overview

Amharic is spoken by over 57 million people, yet it remains significantly underserved by natural language processing tools. This project builds a sentiment analysis model specifically for Amharic text — useful for understanding public opinion on social media, product reviews, or news commentary in Ethiopia.

The model uses a bidirectional LSTM architecture trained on a custom-curated Amharic dataset of social media posts. It incorporates an Amharic-specific tokenizer and achieves 85% accuracy on the test set.

Key Features

  • Bi-directional LSTM model tailored for Amharic text sequences
  • Custom Amharic tokenizer handling Ethiopic script (Fidel alphabet)
  • Dataset of 3,000+ manually labeled Amharic social media posts
  • 3-class sentiment classification: Positive, Negative, Neutral
  • 85% overall accuracy with F1-score of 0.83
  • Text preprocessing: noise removal, normalization, stop-word filtering
  • Confusion matrix and classification report visualization

What I Learned

Building this project gave me hands-on experience with NLP preprocessing for non-Latin scripts, sequence modeling with LSTMs, handling imbalanced datasets using oversampling techniques, and evaluating classification models with precision, recall, and F1-score. It also inspired me to contribute to low-resource language AI research.
